**Numerical Analysis: Numerical Differentiation and Integration (C++)**

This project implements several numerical differentiation and numerical integration techniques in C++ to approximate derivatives and definite integrals while comparing the numerical results with their exact analytical solutions.

The project is divided into two main parts:

**Numerical Differentiation** using finite difference methods.

**Numerical Integration** using Composite Trapezoidal and Composite Simpson's Rules


**Implemented Methods**

**Numerical Differentiation**

Forward Difference

Backward Difference

Centered Difference

For both first derivative and second derivative


**Numerical Integration**

Composite Trapezoidal Rule

Composite Simpson's Rule


**Features**

Pure C++ implementation using the Standard Library

Computes exact analytical derivatives for comparison

Calculates absolute error for every numerical approximation

Displays well-formatted result tables

Implements both differentiation and integration algorithms

Educational implementation for learning numerical analysis techniques


**Problems Solved**

**1. Numerical Differentiation**

The function: f(x) = e^x / x is evaluated over the interval [1, 2] using n = 10 h = 0.1

**The program computes:**

Exact first derivative

Exact second derivative

Forward Difference approximation

Backward Difference approximation

Centered Difference approximation

Absolute error for each method


**2. Numerical Integration**

The program approximates ∫₀⁴ x e^(x²) dx using

Composite Trapezoidal Rule

Composite Simpson's Rule

with n = 40

The exact integral is ½(e¹⁶ − 1) and the approximation error is computed for each numerical method.


**Project Structure**

Compute exact derivatives

Forward Difference (1st derivative)

Backward Difference (1st derivative)

Centered Difference (1st derivative)

Forward Difference (2nd derivative)

Backward Difference (2nd derivative)

Centered Difference (2nd derivative)

Composite Trapezoidal Rule

Composite Simpson's Rule

Display results and errors
